# Find Your Program

A smart, student-focused web platform designed to help users discover the best post-secondary programs in Ontario using AI-powered matching, personalized search history, and a friendly user experience.  
The system includes both the public student interface and a full administrative backend for managing donations, sponsorships, and support tickets.

## ğŸ“˜ Project Description

**Find Your Program** helps students quickly explore academic programs by using an AI-powered recommendation engine that matches their interests and preferences to suitable programs.

### Student Features
- AI-generated program recommendations  
- Save favorite programs  
- View and revisit previous searches  
- Export personal data  
- Submit support requests  

### Admin Features
- Manage donations  
- Manage sponsorship campaigns  
- Respond to student support tickets  
- Approve and manage users  

Built with a modern tech stack, the platform is designed to be fast, secure, and easy to maintain.
